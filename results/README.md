# Results Directory Layout

- `comprehensive_eval/`
  - `figures/`: confusion matrices, ROC/PR, reliability, failure heatmap
  - `tables/`: metrics tables, CIs, calibration, summaries
  - `gradcam/`: representative Grad-CAM visualizations (correct/misclassified)
- `reports/`
  - `research_grade_report.md`: consolidated report summary
- `run_logs/`
  - `legacy_nbconvert/`: initial notebook execution logs
  - `research/`: research-run logs
- `archive/`
  - retained legacy placeholder artifacts
